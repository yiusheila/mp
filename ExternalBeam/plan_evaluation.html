Ways to evaluate a plan
  1. Dose Distribution
    Isodose line/normal tissue hot spot/PTV cold spot
  2. DVH
  3. Plan Quality Index
    e.g. conformity index/homogeneity index
  4. Biological Performance Index
    EUD/TCP/NTCP
Cumulative DVH Dn% = the dose received by n% of the specified volume
Conformity Index (RTOG) Conformity Index
\[ CI = \frac{V_{RI}}{TV} \]
RI = reference isodose
TV = target volume

Other indicators:
(SALT) Lesion Coverage Factor \[ LCF = \frac{TV_{RI}}{TV} \]
(Lomax) Healthy Tissue Conformity Index \[ HTCI = \frac{TV_{RI}}{V_{RI}} \]
Homogeneity Index (ICRU-83) \[ HI = \frac{ D_{2\%} - D_{98\%} }{ D_{50\%} } \]
Clinical/Biological Evaluation Tumour Control Probability (TCP)
Normal Tissue Complication Probability (NTCP)
