Eclipse Operation

Generate Plan Without CT
  1. Open the patient
  2. Go to
    plan parameters
    and insert a new course, plan and field
    (Must be done in plan parameters as there is no 3D image to place the plan on, so it cannot be created in Eclipse.)
  3. Save
  4. Go to external beam planning
  5. Select the plan and create a verification plan
    Planning > create verification plan
Create Water Phantom

  2. Structure > New Phantom Image
    (according to MyVarian - 000006920 - How to create a Water Phantom in Eclipse,
    when after a sphere phantom, need to clear the body structure and use brush to re-draw it... WHY?)
Generate DRR External Beam Planning > Open a plan with CT > right click Field > New DRR
Registration Line Hide/Display When registration lines disappear, right click the small square above the image volumes
uncheck all hide registration options
GPU Memory Limit * Error when trying to start optimisation

  • Remove the SETUP fields
    (most effective)
  • Reduce calculation volume
  • Reduce field size
  • Delete unnecessary structures
    • (less desirable)
  • Reduce structure resolution from high to standard
  • Increase calculation grid size
  • Create 3D volume with fewer CT slices
  • ? use Acuros XB Field dose instead of Plan dose option for intermediate dose
