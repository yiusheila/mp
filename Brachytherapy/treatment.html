(On IGBT 1st Treatment Day)
  1. Insert applicator
     →  insert obturator (metal) in needle to strengthen the needle for easy insertion
     →  fill the bladder with 30 cc saline + 7 cc contrast balloon
     →  insert x-ray marker to 3 tubes
     →  place the needle at the designated position but not pushing beyond the ovoid plane
     →  * apply water-like gel to the applicator parts, ESPECIALLY the holes on ovoids
        so that the ovoid orientation and position show more clearly on MRI
     →  assemble the 3 applicator pieces in patient and lock then in place
     →  add gauze in between tubes and needles
     →  use fixation clip to secure all parts of the applicator
     →  stitch the fixation clip to the patient
     →  draw a marker on patient skin to indicate the position of the fixation clip
  2. AP+LAT X-ray with mobile C-arm
    Make sure the needle insertion lengths are correct
    and the applicator orientation is correct
  3. CT Sim
    Insert x-ray markers to all 3 tubes and all inserted needles
    Load CT onto TPS
    Draw the needles as ROI
  4. MR
    Insert MR markers into 3 tubes (the needles are too thin to insert MR marker)
    Scan 1 x true patient coordinate T2 for planning
    Scan 3 x oblique view T2 (TRA+SAG+COR) aligning the coordinate to the IU tube
       (or for axial scan, perpendicular to IU tube, i.e. parallel to ovoid flange plane)
       > oblique view scan having the IU tube on the same slice
       > easier to reconstruct the applicator
  5. Planning
    (currently still using TG-43, i.e. assuming all surrounding tissue = water, fully scattered)
    (TG-186 is a newer guideline using patient CT for dose calculation)
    (Planning software QEH uses: Oncentra)
     →  load the MRI in true patient coordinate
     →  identify the applicator and needles (catheters)
     →  align the applicator coordinate axes to the applicator IU tube
        make sure the applicator model matches the markers and gel in empty ovoid holes
     →  apply library plan
     →  apply standard dwell points on catheters
        e.g. 1 dwell point at every 2 mm
        offset 5 mm from catheter tip
     →  define point A, point B, rectal point and bladder point for dose taking
        * remember to adjust point B according to patient-applicator rotation
     →  adjust dwell point weighting to ↑ PTV coverage
    ↓ OAR dose
